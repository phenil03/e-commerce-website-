## FUNCTIONS
### Customer
- Customer can view/search products without login.
- Customer can add/remove products to cart without login.
- Real-time search autocomplete and category filtering.
- Persistent shipping address management.
- Secure, theme-aligned checkout process.
- Order tracking with return policy (7-day window).
- Downloadable invoices for each order.
- Send feedback to admin.

---
### Admin
- Unified Analytics Dashboard: View registered customers, product sales, and revenue at a glance.
- Manage Products: Add, edit, and delete products with image support.
- Order Management: View, delete, and update order status (Pending, Confirmed, Delivered).
- Sales Reports: Generate analytics for different periods.
- Feedback Management: View customer feedback directly in the dashboard.

## HOW TO RUN THIS PROJECT
This project is optimized for Python 3.10+ and Django 4.2.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/BLACKDEVIL4114/e-commerce-website-.git
   cd e-commerce-website-
   ```

2. **Setup Virtual Environment**:
   ```bash
   # Windows
   py -3 -m venv .venv
   .venv\Scripts\activate
   ```

3. **Install Dependencies**:
   ```bash
   pip install -r requirement.txt
   ```

4. **Run Migrations & Start Server**:
   ```bash
   python manage.py migrate
   python manage.py runserver
   ```

5. **Access the App**:
   Visit [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
